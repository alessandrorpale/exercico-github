Olá pessoal esse é um treinamento de github

Para criar um repositorio local use git init

Use git add . para adicionar os arquivos

Use git commit -m para comitar as alteracoes

Use git branch -M para alterar o nome da branch principal para main ou deixe como master

Use git remote add origin com a url do remoto para vincular o local com o remoto

Use git push origin para empurrar seu projeto para o repositorio remoto

Resumindo - criar e conectar seu repositório local com o repositório remoto por meio dos comandos git init, git add, git commit, git remote add e git push
